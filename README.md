# Inventory SaaS Demo

A production-grade Inventory Management SaaS application built with **React + Vite + TailwindCSS**.  
This project showcases authentication, protected routes, dark mode toggle, and a clean SaaS-style UI.

---

## Features
- Authentication (Login / Signup / Logout)
- Dashboard with summary cards
- Products Management
- Stock Tracking
- Dark Mode Toggle
- Responsive UI with TailwindCSS
- Protected Routes for logged-in users

---

## Tech Stack
- Frontend: React, Vite, React Router
- Styling: TailwindCSS
- State Management: Context API
- Auth: Custom AuthProvider
- Deployment: GitHub Pages / Vercel / Netlify
